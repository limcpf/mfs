# About

This is a file-system style blog that displays markdown notes as if they were files in a directory tree.

## Features

- **File Tree Navigation** - Browse documents like a file explorer
- **Real Path Routing** - Each document has a real URL path
- **Syntax Highlighting** - Code blocks are highlighted with Shiki
- **Wikilinks** - Obsidian-style [[wikilinks]] are supported
